插入图片

  1. 安装插件npm install https://github.com/7ym0n/hexo-asset-image --save
  2. 在根目录配置文件_config.yml中的post_asset_folder:false选项设置为true.
  3. 在xxx.md中引用图片

列表

使用 * + - 表示无序列表

  • 计划A
    • A计划第一步
      • 首先XXX

    • B计划第一步

      表格

      使用|来分隔不同的单元格，使用-来分隔表头与其他行
      名字 性别 年龄
      张三 男 18
      李四 女 20

注意：为了使 Markdown 更清晰，|和-两侧需要至少有一个空格（最左侧和最右侧的 | 外不需要）

  • :— 代表左对齐
  • :–:代表居中对齐
  • —:代表右对齐

例如：

名字 性别 年龄

    创建环境

    CPU环境

    • conda create -n tf-cpu pip python=3.7

    • 激活环境

      • conda activate tf-cpu

    • 修改pip.ini，换源

    • 安装Tensorflow

      • pip install tensorflow -i http://pypi.douban.com/simple --trusted-host pypi.douban.com
      • pip install --index-url https://mirrors.aliyun.com/pypi/simple/ tensorflow
      • 或者pip install --index-url https://pypi.douban.com/simple tensorflow

    • 检验是否安装成功

      • 输入python，回车
      • import tensorflow as tf，回车
      • print(tf.reduce_sum(tf.random.normal([100,100]))),回车
      • 正常输出则安装成功
